# User Guide

The **RI-SCALE Model Hub** is a platform for sharing and discovering standardized AI models for research infrastructures. It allows you to:
1.  **Search and download** open-source models.
2.  **Upload and share** your own models.

## Browsing Models

1.  **Search**: Use the search bar on the home page to find models by name, tags, or description.
2.  **Filter**: You can filter models by tags such as task (e.g., segmentation, restoration) or modality.
3.  **Details**: Click on any model card to view detailed information, including:
    *   Description and authors
    *   Version history
    *   Citations
    *   File downloads

## Uploading a Model

To contribute a model to the hub:
1.  Click **Upload** in the navigation bar.
2.  **Log in** with your credentials (via Hypha).
3.  Fill in the required metadata:
    *   **Name**: A descriptive name for your model.
    *   **Description**: What does your model do?
    *   **Tags**: Add relevant tags to make it discoverable.
    *   **Files**: Upload your model weights and configuration files.
4.  Submit your model. It will be available for others to discover once processed.
