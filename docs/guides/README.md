# Guides
The BioImage Model Zoo is intended to be used by different roles in the community as stated in [How to participate in the BioImage Model Zoo](/getting_started/README.md). This section contains guides for the different roles in the community.

* **[User Guide](user_guide.md)**: Guides for users willing to download and use the models and other resources from the BioImage Model Zoo
* **[Developers Guide](developer_guide.md)**: Guides for who is willing to contribute to the BioImage Model Zoo with models, applications or softwares.
* **[Community Partners Guide](community_partners_guide/README.md)**: If you want to join as a Community Partner, this is your place to visit.
